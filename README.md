# 🎓 Student Marks Analysis using Python & Pandas

## 📌 Project Overview
This project is a simple **Student Marks Analysis System** built using **Python** and the **Pandas** library. It analyzes students' marks in multiple subjects, calculates total marks, percentage, assigns grades, identifies the topper, computes subject-wise averages, and filters students scoring above 85%.

---

## 🚀 Features

- Create a student marks dataset using Pandas.
- Calculate total marks for each student.
- Calculate percentage.
- Assign grades based on percentage.
- Find the topper of the class.
- Calculate subject-wise average marks.
- Display students scoring above 85%.
